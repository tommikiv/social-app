# Social App
Made with React Native using Firebase/Firestore for authentication and storing posts.
Create your own posts and scroll the feed to see realtime updates!

# Features
- Email/Password Authentication
- Create Posts
- Realtime Feed Updates

# Setup
- npm install
- create a Firebase project & app [Here](https://firebase.google.com)
- make a .env file and add the needed Firebase config variables (see .env.example)
- npm start
- use Expo Go to run the app on your phone

# Note
This is a simple demo app for learning. Do not use as-is in production.
